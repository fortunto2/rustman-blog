# CLAUDE.md — rustman-blog

Personal build-in-public blog. Astro 5 static site deployed to Cloudflare Pages.

## Stack

- **Framework:** Astro 5 (static output)
- **Styling:** Tailwind CSS 4
- **Markdown:** markdown-it + highlight.js + [[wikilinks]]
- **Search:** Pagefind (client-side, indexed at build)
- **Deploy:** Cloudflare Pages (free tier, auto-build on push)
- **Domain:** rustman.org

## Content Pipeline

Content lives in `../wiki/` (solopreneur KB). Blog pulls only `publish: true` pages.

```
../wiki/*.md → collect-content.ts → src/content/{articles,projects,notes}/ → Astro build → dist/
```

To publish a wiki page: add `publish: true` to its frontmatter. Optional: `publish_as: project` or `publish_as: note` (default: article).

## Commands

```bash
pnpm collect    # Pull publish:true pages from ../wiki/
pnpm dev        # Dev server :4321
pnpm build      # Astro build + Pagefind index
pnpm deploy     # Build + wrangler pages deploy
```

## Structure

```
blog/
  src/
    lib/config.ts          # Site config (domain, nav, social)
    lib/md.ts              # Markdown pipeline (wikilinks, frontmatter, render)
    lib/content.ts         # Content loading from src/content/
    layouts/BaseLayout.astro
    pages/
      index.astro          # Homepage (recent posts)
      articles/[slug].astro
      projects/[slug].astro
      notes/[slug].astro
      about.astro
      search.astro         # Pagefind UI
      rss.xml.ts
      llms.txt.ts
    styles/global.css      # Tailwind + theme variables
    content/               # Collected from wiki (gitignored in blog, committed for deploy)
      articles/*.md
      projects/*.md
      notes/*.md
  scripts/
    collect-content.ts     # Reads ../wiki/, filters publish:true
  astro.config.mjs
  wrangler.toml
```

## Routes

```
/                    → homepage (recent posts)
/articles/{slug}     → article (wiki concept/summary pages)
/projects/{slug}     → project update (build-in-public)
/notes/{slug}        → short note
/about               → about page
/search              → Pagefind search
/rss.xml             → RSS feed
/llms.txt            → AI crawler content map
/sitemap-index.xml   → auto-generated
```

## Workflow

1. Edit wiki pages in solopreneur (Obsidian, Claude Code, etc.)
2. Add `publish: true` to frontmatter of pages to publish
3. `cd blog && pnpm collect && pnpm build` — or just `pnpm collect` then commit
4. `git push` → CF Pages auto-deploys

## Frontmatter for publishing

```yaml
---
type: concept
title: "Page title"
created: 2026-04-07
tags: [tag1, tag2]
publish: true              # required to appear on blog
publish_as: article        # optional: article (default) | project | note
---
```

## Don't

- Don't edit `src/content/` manually — it's generated by `pnpm collect`
- Don't commit solopreneur private content (CRM, visa, personal tagged pages)
- Don't add SSR — static is simpler, free, faster
