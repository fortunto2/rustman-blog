# awesome-rust

The canonical list of Rust code and resources. 57k stars, community-maintained.

## Structure

Three top-level sections:

- **Applications** (25+ categories): audio, blockchain, databases, games, graphics, system tools, web servers
- **Development Tools** (11 categories): build systems, debugging, IDEs, testing, profiling
- **Libraries** (40+ categories): AI/ML, crypto, data structures, networking, GUI, web frameworks

## How to use

Starting a new Rust project? Check the relevant category before picking a crate. The list shows build status, so you can spot abandoned projects.

Notable entries: Servo, Deno, SurrealDB, Alacritty, ripgrep, fd, bat, Zellij, Helix.
